Make your google maps more interesting.

== Description ==

This plugin uses google maps, with the integration of FourSquare API, helps you to make your contact us page more interesting. It helps you to show the latest 3 reviews/tips by your customer, and will also show nearby venues.

== Installation ==
1. Download and extract the plugin
2. Upload the plugin folder(*foursquaregoogle*) to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Place [4sqrgo] inside your post/page content.
